# Herbier Gourmand — édition quotidienne

Cette version contient 40 recettes et des fonctions pour cuisiner tous les jours :

- choix rapide : rapide, économique ou végétarien ;
- surprise du jour ;
- recherche par ingrédient ;
- garde-manger avec plusieurs ingrédients ;
- favoris ;
- export PDF de la sélection ou du volume complet ;
- fonctionnement hors connexion après installation comme application web.

Ce paquet reste un projet web Android/PWA, pas encore un APK signé.
